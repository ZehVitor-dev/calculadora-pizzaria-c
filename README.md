# Calculadora de conta para pizzaria (C)

Programa em linguagem C desenvolvido para a faculdade.

## Funcionalidades

- Lê o valor total da conta da mesa
- Lê a quantidade de pessoas
- Aplica um percentual de desconto para o estabelecimento
- Mostra:
  - Valor do desconto
  - Total com desconto
  - Valor a pagar por pessoa

## Como compilar e executar

```bash
gcc main.c -o calculadora_pizzaria
./calculadora_pizzaria
